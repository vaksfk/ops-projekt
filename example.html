• Einfaches Simplex Beispiel

    -7x1 - 8x2 - 10x3 + P = 0

    2x1 + 3x2 + 2x3 + s1 = 1000

    x1 + x2 + 2x3 + s2 = 800

  • Schritt 1

    x1 x2 x3 s1 s2 P
    s1 2 3 2 1 0 0 1000
    s2 1 1 2 0 1 0 800
    -7 -8 -10 0 0 1 0

    Kleinste Wert der Zielfunktion ist -10
    Auswahl der Pivotzeile: 800/2 = 400
    0.5 * R1 => R2

  • Schritt 2

    x1 x2 x3 s1 s2 P
    s1 2 3 2 1 0 0 1000
    s2 0.5 0.5 1 0 0.5 0 400
    -7 -8 -10 0 0 1 0

    Kleinste Wert der Zielfunktion ist -10
    Auswahl der Pivotzeile: 400/1 = 400
    -2 * R2 + R1 => R1
    10 * R2 + R3 => R3

  • Schritt 3

    x1 x2 x3 s1 s2 P
    s1 1 2 0 1 -1 0 200
    s2 0.5 0.5 1 0 0.5 0 400
    -2 -3 0 0 5 1 4000

    Kleinste Wert der Zielfunktion ist -3
    Auswahl der Pivotzeile: 200/2 = 400
    0.5 * R1 -> R1

  • Schritt 4

    x1 x2 x3 s1 s2 P
    s1 0.5 1 0 0.5 -0.5 0 100
    x3 0.5 0.5 1 0 0.5 0 400
    -2 -3 0 0 5 1 4000

    Kleinste Wert der Zielfunktion ist -13
    Auswahl der Pivotzeile: 100/1 = 100
    -0.5 * R1 + R2 => R2
    3 * R1 + R3 => R3

  • Schritt 5

    x1 x2 x3 s1 s2 P
    x2 0.5 1 0 0.5 -0.5 0 100
    x3 0.25 0 1 -0.25 0.75 0 350
    -0.5 0 0 1.5 3.5 1 4300

    Kleinste Wert der Zielfunktion ist -0.5
    Auswahl der Pivotzeile: 200/1 = 200
    R1 * 2 => R1

  • Schritt 6

    x1 x2 x3 s1 s2 P
    x2 1 2 0 1 -1 0 200
    x3 0.25 0 1 -0.25 0.75 0 350
    -0.5 0 0 1.5 3.5 1 4300

    Kleinste Wert der Zielfunktion ist -0.5
    Auswahl der Pivotzeile: 200/1 = 200
    -0.25 * R1 + R2 => R2
    0.5 * R1 + R3 => R3

  • Schritt 7

    x1 x2 x3 s1 s2 P
    x1 1 2 0 1 -1 0 200
    x3 0 -0.5 1 -0.5 1 0 300
    0 1 0 2 3 1 4400

    Fertig, es gibt keine negative Zahlen in der Zielfunktion mehr.
    x1 = 200
    x3 = 300
    x2, s1, s2 = 0
    MAX = 4400
